Requisitos:
    . nodeJS
    . npm
    . expo-cli ( npm i --location=global expo-cli )

    Ademas, hace falta instalar la aplicacion de EXPO GO en
    el dispositivo movil con el que querramos probar
************************************************
Una vez cumplidos los requisitos, correr
    . npm install
    . expo start
